# 🔥 GoldenHeat — AI 中长周期投资决策系统

> 用 AI 做月线级别的趋势判断，不做日线赌博。

## 项目定位
GoldenHeat 是一个面向 **中长周期投资决策** 的分析面板。核心不是做短线交易提示，而是把宏观周期、市场热度、资产配置、月线信号和牛熊仓位集中到一个统一的 dashboard 里，帮助判断：

- 当前处于什么经济阶段
- 该超配什么资产
- 哪些标的处于更好的月线位置
- 当前仓位应该更进攻还是更保守

---

## 核心模块
- **美林时钟**：判断宏观周期与推荐资产
- **市场热度**：用 0–100 温度刻画市场冷热
- **资产配置**：按当前周期给出配置方向
- **月线信号**：基于 MA5 / MA10 / MA20 等结构识别趋势与回调
- **牛熊分界**：结合年线 / 两年线做仓位建议

---

## 技术架构
```text
数据采集层 → 分析引擎 → 信号生成 → 展示层
  │              │            │          │
  ├─宏观经济数据  ├─美林时钟模型  ├─月线信号   ├─Web Dashboard
  ├─月线K线数据   ├─趋势分析      ├─牛熊判断   ├─Telegram Bot
  ├─利率/CPI/PMI ├─偏离度计算     ├─偏离度警报  └─邮件周报
  └─估值数据     └─LLM辅助研判    └─持仓建议
```

---

## 技术栈
- 后端：Python + FastAPI
- 数据：SQLite + 定时采集
- 分析：pandas + numpy
- 前端：React + Vite + Recharts
- AI：LLM 辅助宏观研判与信号解读
