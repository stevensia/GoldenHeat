# 🔥 GoldenHeat — AI 中长周期投资决策系统

> **用 AI 做月线级别的趋势判断，不做日线赌博**

## 核心理念

- 美林时钟 + 宏观经济周期 → 判断大方向（做什么资产类别）
- 月线级别回调/反弹信号 → 判断标的入场/出场时机
- 牛熊分割线 → 判断年线级别仓位（满仓/半仓/空仓）
- **周线日线不做讨论，等于赌博**

## 技术架构

```
数据采集层 → 分析引擎 → 信号生成 → 展示层
  │              │            │          │
  ├─宏观经济数据  ├─美林时钟模型  ├─月线信号   ├─Web Dashboard
  ├─月线K线数据   ├─趋势分析    ├─牛熊判断   ├─Telegram Bot
  ├─利率/CPI/PMI ├─偏离度计算   ├─偏离度警报  └─邮件周报
  └─估值数据     └─LLM 辅助研判 └─持仓建议
```

## 技术栈

- **后端**: Python + FastAPI
- **数据**: SQLite + 定时采集
- **分析**: pandas + numpy + ta-lib
- **AI**: LLM 辅助宏观研判 + 信号解读
- **前端**: React (Vite) 仪表盘
- **通知**: Telegram Bot + 邮件

## 开发阶段

- **Phase 1**: 数据采集 + 美林时钟引擎
- **Phase 2**: 月线信号系统 + 牛熊分割线
- **Phase 3**: Web Dashboard + 通知系统
- **Phase 4**: LLM 辅助研判 + 回测验证
